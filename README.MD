# Project Title
Manage a Data base of customers.

# Description
this program will read pre made customers from a csv file,
the user will be able to add new customers or to follow their
actions inside with the operators command ("!=,=,<,>).

# How To Use
Step #1: run command:
make
Step #2: run command:
./main customers.txt

to delete .o files
run command:
make clean
